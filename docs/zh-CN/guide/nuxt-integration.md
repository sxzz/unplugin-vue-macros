# Nuxt 整合

### 安装

::: code-group

```bash [npm]
npm i -D @vue-macros/nuxt
```

```bash [yarn]
yarn add -D @vue-macros/nuxt
```

```bash [pnpm]
pnpm add -D @vue-macros/nuxt
```

:::

## 配置

```ts
// nuxt.config.ts
export default {
  modules: [
    '@vue-macros/nuxt',
    // ...
  ],
  macros: {
    // configure plugin options, if needed
  },
}
```
